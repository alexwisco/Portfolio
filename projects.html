Digital Synth - Raspberry Pi

Developed and built a sine wave musical synthesizer in Python using a Raspberry Pi 4. Constructed and wired a 12-key layout piano with a breadboard and mapped modular combinations of sine wave frequencies to each button. Incorporated user-friendly design from octave switching, waveform shifting buttons, and more

Music Taste Diversity Quantifier

Developed a Python machine learning program to quantify the diversity of a user's Spotify artists. Utilized pandas library and Spotify's Spotipy library along with Scikit-Learn to cluster songs based on Spotify's analytic metrics. Processed data from Spotify using their API to generate real-world observable results.

Convex Hull Graham Scan Visualizer

HTML, CSS, JS interactive webpage displaying the steps of the Graham Scan algorithm to find the convex hull of a set of points.
